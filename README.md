# Alchemlyb-YANK
A utility script to convert YANK NetCDF4 output files into a format compatible with Alchemlyb for free energy analysis. The ABFE Alchemlyb workflow has also been modified to ensure compatibility with the converted data.
### Usage
The script converts YANK NetCDF4 output files into a format compatible with Alchemlyb and performs the ABFE analysis. The complex and solvent legs are analyzed independently and combined to obtain the final binding free energy.
### Command line arguments
```
-f, --input        NetCDF4 input file generated by YANK.
-n, --njobs        Number of parallel jobs used for the analysis.
-T, --temperature  Simulation temperature (in Kelvin).
-s, --skiplambda   Lambda state(s) to skip during the analysis (optional).
-o, --output       Output file name.
```
### Example
```
python Analysis_ALCHEMLYB.py \
    -f analysis.nc \
    -n 8 \
    -T 298.15 \
    -s 46 \
    -o results.txt
```
`--skiplambda` can be used to remove problematic lambda windows from the analysis if convergence issues are detected.
### Output
The script prints the free energy contributions from the complex and solvent legs, applies the standard state correction, and reports the final binding free energy:
```
Results:
    Complex leg:   ΔG_complex ± error
    Solvent leg:   ΔG_solvent ± error
    Standard correction: value (kT and kcal/mol)
    Final result:  ΔG_bind ± error | Kd (µM)
```
Where:
+ **ΔG_complex** – free energy from the complex leg
+ **ΔG_solvent** – free energy from the solvent leg
+ **Standard correction** – standard state correction applied to the ABFE calculation
+ **ΔG_bind** – final binding free energy
+ **Kd (µM)** – estimated dissociation constant calculated from ΔG

The final ΔG is computed as:
`ΔG = ΔGsolvent − ΔGcomplex + standard_state_correction` 
and the dissociation constant (Kd) is estimated from ΔG at the specified temperature.
